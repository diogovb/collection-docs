# Criando um projeto

O Studio é onde tudo se junta: cenas, renders, apresentação, especificação e compartilhamento com a cliente. Cada projeto é um container para um trabalho.

## Como criar

<Steps>
  <Step title="Acesse o Studio">
    Em [app.collection.com.br](https://app.collection.com.br), clique em **Studio** no menu.
  </Step>
  <Step title="Clique em 'Novo projeto'">
    Dê um nome ao projeto (ex: "Apartamento 302 — Mariana").
  </Step>
  <Step title="Adicione os dados">
    Nome da cliente, capa do projeto — tudo visível quando você compartilhar.
  </Step>
</Steps>

## O Dashboard do projeto

Dentro de cada projeto, você acompanha:
- **Ambientes** e **produtos especificados**
- **Valor total** do projeto
- **Recompensa estimada** — quanto você pode ganhar com as compras da cliente
- **Categorias** — gráfico mostrando distribuição (Móveis, Materiais, Esquadrias...)
- **Social** — curtidas, comentários e visualizações (se publicar na comunidade)

<Tip>
Publicar o projeto na **comunidade** rende **+10 Collecs** de bônus.
</Tip>
