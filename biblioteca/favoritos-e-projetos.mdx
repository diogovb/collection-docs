# Favoritos e adicionar ao projeto

## Favoritos ❤️

Encontrou um bloco que não vai usar agora? Clique no **coração** no card do produto. Ele fica salvo na sua lista de favoritos.

Seus favoritos **sincronizam entre web e plugin** — salve na web e acesse no SketchUp, e vice-versa.

## Adicionar ao projeto

No card de cada produto, clique em **"Adicionar ao projeto"**:
1. Selecione o projeto e o ambiente
2. O produto vai para a Especificação do projeto
3. Fica disponível para tagear na Apresentação

O número entre parênteses (ex: "(82)") mostra quantos profissionais já adicionaram aquele produto — é uma boa referência de popularidade.

<Tip>
Adicionar produtos ao projeto é essencial para a Especificação e as Recompensas. Quanto mais completa a lista, mais profissional a entrega para sua cliente.
</Tip>
