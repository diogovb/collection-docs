## Pelo Plugin (recomendado)

A forma mais rápida: clique no bloco na Biblioteca do plugin e ele é **inserido direto na sua cena**. Materiais são aplicados automaticamente.

### Onde ficam os mapas

Quando você baixa materiais (.skm) pelo plugin, os mapas de textura ficam em:

```
C:/collection/maps
```

<Warning>
Não mova ou delete essa pasta. Os materiais do SketchUp referenciam os mapas nesse local.
</Warning>

## Pela Web

Na web ([app.collection.com.br](https://app.collection.com.br) → Biblioteca):
- **Blocos 3D** → download em `.skp`
- **Materiais** → download em `.zip` com todos os mapas de textura
