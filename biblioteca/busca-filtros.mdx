# Busca e filtros

## Busca

Digite no campo de busca por:
- **Nome do produto** — "Cadeira Saragoça"
- **Marca** — "Deca", "Docol", "Portobello"
- **Tipo** — "pendente", "porcelanato", "sofá"

## Filtros

Clique no ícone de **filtros** para refinar por categoria.

## Ordenação

Use o botão de ordenação para organizar por:
- **Mais recentes** — os últimos blocos adicionados
- Outras opções disponíveis conforme a categoria

<Tip>
Não encontrou o que procura? Use o botão **📢 Sugerir produtos** no canto inferior da Biblioteca. A equipe analisa e pode adicionar à biblioteca.
</Tip>
