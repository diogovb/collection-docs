# Instalando o plugin do Collection

Tudo começa aqui. O plugin conecta o SketchUp ao Collection — é por ele que você baixa blocos, envia cenas e renderiza.

## O que você precisa

| Requisito | Versão |
|---|---|
| **Sistema** | Windows 10 ou superior |
| **SketchUp** | 2024, 2025 ou 2026 |
| **V-Ray** | 6.20.03 (se usar) |
| **Enscape** | 4.3.0+ (se usar) |

<Warning>
O plugin está disponível **apenas para Windows** no momento. Se você usa Mac, pode usar a [Biblioteca pela web](https://app.collection.com.br) e fazer upload direto no Studio.
</Warning>

## Passo a passo

<Steps>
  <Step title="Baixe o plugin">
    Acesse [collection.com.br/instalar](https://www.collection.com.br/instalar) e clique em **"Clique aqui para fazer o download"**. O arquivo é um `.rbz` (Collection Plugin 2.2.0.17).
  </Step>
  <Step title="Desinstale a versão anterior (se tiver)">
    No SketchUp: **Janela → Gerenciador de Extensões** → encontre "Collection" → **Desinstalar**.
  </Step>
  <Step title="Reinicie o SketchUp">
    Feche e abra o SketchUp. Importante para limpar a versão anterior.
  </Step>
  <Step title="Instale o arquivo .rbz">
    No SketchUp: **Janela → Gerenciador de Extensões → Instalar Extensão** → selecione o `.rbz`.
  </Step>
  <Step title="Pronto! ✨">
    Dois botões aparecem na barra de ferramentas:
    - 📚 **Biblioteca** — buscar e baixar blocos
    - 🎨 **Studio** — enviar cenas e renderizar
  </Step>
</Steps>

<AccordionGroup>
  <Accordion title="O plugin não aparece depois de instalar">
    Reinicie o SketchUp. Confira em **Janela → Gerenciador de Extensões** se está ativado.
  </Accordion>
  <Accordion title="Uso Mac, como faço?">
    Use a Biblioteca pela web em [app.collection.com.br](https://app.collection.com.br) e faça upload de imagens direto no Studio.
  </Accordion>
</AccordionGroup>
