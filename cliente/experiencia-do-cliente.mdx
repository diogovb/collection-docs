# A experiência do cliente

O Collection não é só uma ferramenta pra você projetar. É um **ecossistema que conecta seu projeto à compra**. Quando seu cliente compra um produto pelo seu link, você ganha.

## Como funciona

```
Você projeta → Compartilha → Cliente vê e compra → Você ganha 5%
```

É simples: o projeto que você já faz no dia a dia agora **gera receita**. Cada produto tageado e especificado é uma oportunidade de venda.

## O que o cliente vê

Quando você compartilha o projeto, seu cliente recebe uma **experiência interativa**:

- 🖼️ **Imagens do projeto** divididas por ambiente
- 🏷️ **Produtos de cada ambiente** com nome, marca e preço
- 🛒 **Carrinho de compras** — adiciona o que quiser
- 💳 **Checkout completo** — compra direto pela experiência

Não é um PDF estático. É uma experiência de compra pensada para converter.

## Por que isso importa

- seu cliente não precisa sair procurando produtos em lojas
- ele compra exatamente o que você especificou — sem substituições
- O projeto chega inteiro — como você imaginou
- E você é **recompensado** por cada compra

<Tip>
Quanto mais completa a especificação e mais bonita a apresentação, maior a chance do cliente comprar pelo seu link. Cada detalhe conta.
</Tip>
