# Recompensas — 5% por compra 💰

Cada produto que seu cliente compra pelo link do projeto gera **5% de recompensa** para você. É uma receita extra que vem do trabalho que você já faz.

## Como funciona

<Steps>
  <Step title="Você especifica">
    Adicione produtos reais da Biblioteca ao projeto.
  </Step>
  <Step title="Você compartilha">
    Envie a experiência de compra para seu cliente.
  </Step>
  <Step title="o cliente compra">
    O cliente navega, escolhe e faz o checkout.
  </Step>
  <Step title="Você ganha">
    5% do valor de cada produto comprado vira sua recompensa.
  </Step>
</Steps>

## Exemplo

Se seu cliente comprar R$ 10.000 em produtos pelo seu link, você recebe **R$ 500** de recompensa.

## No Dashboard

Acompanhe em tempo real:
- **Recompensa estimada** — valor potencial de projetos acessados pelo cliente
- **Recompensa efetivada** — valor real de compras concluídas
- **Barra de progresso** — por projeto

## Prazo de saque

A recompensa fica disponível **14 dias após a compra** (prazo de devolução). Se o cliente devolver, a recompensa é cancelada.

## Como sacar

1. Aguarde os 14 dias
2. Emita uma **Nota Fiscal** no valor da recompensa
3. Envie a NF para o Collection
4. Receba via **PIX**

<Tip>
Quanto mais projetos você especifica e compartilha com clientes, mais a recompensa cresce. É uma renda que escala com seu trabalho.
</Tip>
