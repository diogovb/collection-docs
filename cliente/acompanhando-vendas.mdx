## Onde acompanhar

No **Dashboard do Studio** você vê os números gerais:
- Projetos enviados para clientes
- Recompensa estimada (total)
- Recompensa efetivada (total)

Dentro de **cada projeto**, os dados são específicos:
- Valor total dos produtos especificados
- Recompensa estimada daquele projeto
- Barra de progresso do objetivo de recompensa
- Histórico de ações (quando o cliente acessou, comprou, etc.)

## Ciclo de vida da recompensa

```
Projeto enviado → Cliente acessou → Cliente comprou → 14 dias → Saque disponível
```

| Status | Significado |
|---|---|
| **Estimada** | Projeto foi acessado pelo cliente, mas ainda não comprou |
| **Efetivada** | Compra realizada, aguardando prazo de 14 dias |
| **Disponível** | Passaram 14 dias, você pode sacar |

## Como sacar

1. Emita uma **Nota Fiscal** no valor disponível
2. Envie para o Collection
3. Receba via **PIX**
