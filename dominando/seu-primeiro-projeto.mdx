# Seu primeiro projeto

O Studio é onde tudo se junta: renders, apresentação, especificação e compartilhamento. Cada projeto é um container para um trabalho.

## Criando o projeto

<Steps>
  <Step title="Acesse o Studio">
    Em [app.collection.com.br](https://app.collection.com.br), clique em **Studio**.
  </Step>
  <Step title="Clique em 'Novo projeto'">
    Dê um nome (ex: "Apartamento Mariana") e, opcionalmente, o nome da cliente.
  </Step>
  <Step title="Pronto!">
    Seu projeto aparece no Dashboard. Agora é hora de adicionar cenas.
  </Step>
</Steps>

## O que o Dashboard mostra

Dentro do projeto, você vê:
- **Ambientes** — quantos ambientes tem no projeto
- **Produtos especificados** — quantos blocos estão vinculados
- **Valor total** — soma dos produtos com preço
- **Recompensa estimada** — quanto você pode ganhar (5% das compras)
- **Categorias** — gráfico mostrando a distribuição (Móveis, Materiais, etc.)

## Personalize

- Escolha uma **capa do projeto** (clique na imagem de capa para trocar)
- Adicione o **nome da cliente**
- Os dados são visíveis quando você compartilhar com a cliente
