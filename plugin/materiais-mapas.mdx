# Materiais e mapas de textura

## Formatos

| Tipo | Formato | Via Plugin | Via Web |
|---|---|---|---|
| **Blocos 3D** | .skp | Inserido direto na cena | Download manual |
| **Materiais** | .skm | Aplicado automaticamente | Download .zip com mapas |

## Onde ficam os mapas

Quando você baixa materiais pelo plugin, os mapas de textura são salvos em:

```
C:/collection/maps
```

<Warning>
Não mova ou delete essa pasta. Os materiais no SketchUp referenciam os mapas nesse local. Se a pasta sumir, as texturas desaparecem da cena.
</Warning>

## Compatibilidade

Os materiais da Biblioteca Collection já vêm configurados para:

- ✅ **V-Ray** 6.20.03
- ✅ **Enscape** 4.3.0 ou superior

Não precisa ajustar nada — é baixar e usar.
