# Instalando o Plugin do Collection

O plugin do Collection é como tudo começa. Ele conecta o SketchUp ao Collection — biblioteca, render, tudo.

## Requisitos

<Warning>
O plugin está disponível **apenas para Windows**. MacOS não é suportado no momento.
</Warning>

| Requisito | Versão |
|---|---|
| **Sistema** | Windows 10 ou superior |
| **SketchUp** | 2024, 2025 ou 2026 |
| **V-Ray** | 6.20.03 (se usar V-Ray) |
| **Enscape** | 4.3.0 ou superior (se usar Enscape) |

## Passo a passo

<Steps>
  <Step title="Baixe o plugin">
    Acesse [collection.com.br/instalar](https://www.collection.com.br/instalar) e clique em **"Clique aqui para fazer o download"**.

    O arquivo é um `.rbz` (Collection Plugin 2.2.0.17 — pacote completo com Biblioteca 3D + Collection IA + Index).
  </Step>
  <Step title="Desinstale a versão anterior (se tiver)">
    Se você já tinha uma versão do plugin, **desinstale primeiro**:
    - No SketchUp: **Janela → Gerenciador de Extensões**
    - Encontre "Collection" e clique em **Desinstalar**
  </Step>
  <Step title="Reinicie o SketchUp">
    Feche e abra o SketchUp novamente. Isso é importante para limpar a versão antiga da memória.
  </Step>
  <Step title="Instale o novo plugin">
    - No SketchUp: **Janela → Gerenciador de Extensões → Instalar Extensão**
    - Selecione o arquivo `.rbz` que você baixou
    - Confirme a instalação
  </Step>
  <Step title="Pronto!">
    Dois novos botões aparecem na barra de ferramentas do SketchUp:
    - 📚 **Biblioteca** — para buscar e baixar blocos
    - 🎨 **Studio** — para enviar cenas e renderizar
  </Step>
</Steps>

## Problemas comuns

<AccordionGroup>
  <Accordion title="O plugin não aparece depois de instalar">
    Reinicie o SketchUp. Se ainda não aparecer, verifique em **Janela → Gerenciador de Extensões** se o plugin está ativado.
  </Accordion>
  <Accordion title="Uso MacOS, posso instalar?">
    Infelizmente o plugin não está disponível para MacOS no momento. Você pode usar a **Biblioteca pela web** (app.collection.com.br) e fazer upload de cenas diretamente no Studio.
  </Accordion>
  <Accordion title="Minha versão do SketchUp não é suportada">
    O plugin funciona com SketchUp 2024, 2025 e 2026. Versões anteriores não são suportadas.
  </Accordion>
</AccordionGroup>
