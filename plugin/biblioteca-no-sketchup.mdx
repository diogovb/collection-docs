# Biblioteca no SketchUp

Com o plugin instalado, você acessa toda a Biblioteca do Collection sem sair do SketchUp.

## Como usar

1. No SketchUp, clique no botão **📚 Biblioteca** do plugin
2. Busque por nome, marca ou categoria
3. Clique no bloco desejado para ver detalhes
4. Clique em **Baixar** — o bloco é inserido diretamente na sua cena

## Materiais

Quando você baixa materiais (.skm) pelo plugin:
- Os materiais são **aplicados automaticamente** na cena
- Os mapas de textura ficam salvos em `C:/collection/maps`
- Materiais já vêm **configurados para V-Ray e Enscape**

<Tip>
Pela web (app.collection.com.br), materiais são baixados em `.zip` com todos os mapas. Pelo plugin, tudo é automático.
</Tip>
