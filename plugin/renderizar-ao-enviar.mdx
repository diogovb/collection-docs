# Renderizar ao enviar

Em vez de enviar a cena e renderizar depois no Studio, você pode fazer tudo de uma vez — direto do SketchUp.

## Como ativar

Ao enviar a cena pelo botão **Studio** do plugin:

1. Marque a opção **"Renderizar ao enviar"**
2. Configure:
   - **Ambientação (mood)** — o estilo de iluminação e atmosfera
   - **Luzes artificiais** — ative se sua cena tem luminárias
   - **Prompt personalizado** — opcional, para instruções específicas
3. Envie

A cena vai para o Studio e o render começa automaticamente. Em ~30 segundos o resultado aparece na Apresentação.

<Info>
O prompt personalizado é opcional. O Collection já usa um prompt padrão otimizado que funciona bem para a maioria das cenas.
</Info>

## Quando usar

- **Projetos rápidos** — quando quer o render o mais rápido possível
- **Múltiplas cenas** — envie várias cenas de uma vez e todas são renderizadas automaticamente
- **Fluxo otimizado** — evita ter que abrir o Studio para cada render
