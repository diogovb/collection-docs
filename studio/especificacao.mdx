# Especificação (Planilha)

A Especificação é uma tabela com todos os produtos do seu projeto — como uma planilha de especificação técnica.

## O que mostra

- **Lista de produtos** — nome, marca e número da tag
- **Organizado por ambiente**
- **Filtro por categoria** — Todos, Móveis, Esquadrias, Materiais, etc.
- **Resumo de recompensa** — valor total, recompensa estimada e média (5%)

## De onde vêm os produtos

1. **Do SketchUp** — produtos da Biblioteca que estão na cena são importados automaticamente
2. **Adicionados manualmente** — você pode adicionar produtos direto pela Especificação

## Para que serve

- Conferir todos os produtos especificados no projeto
- Calcular valor total e recompensa estimada
- Exportar a lista para apresentar à cliente
- Garantir que nenhum produto ficou de fora
