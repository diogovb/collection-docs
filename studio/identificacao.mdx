# Identificação de produtos

A Identificação permite tagear os produtos direto na imagem da cena. Sua cliente vê exatamente qual produto é cada item.

## Como tagear

1. Na aba **Especificação** do Studio (será renomeada em breve)
2. Clique na imagem onde o produto está
3. Selecione qual produto da lista associar àquele ponto
4. Uma tag numerada (1, 2, 3...) aparece sobre a imagem

## Gerenciar tags

- As tags ficam **numeradas** e vinculadas à lista de produtos
- Botão **"Ocultar/mostrar marcações"** para alternar a visibilidade
- No **Modo apresentação**, as tags ficam visíveis para sua cliente

<Tip>
Tageie os produtos antes de compartilhar com a cliente. Assim ela vê exatamente o que é cada item e pode adicionar ao carrinho.
</Tip>
