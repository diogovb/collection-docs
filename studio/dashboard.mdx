# Dashboard do Studio

O Dashboard é sua página inicial. Aqui você vê todos os seus projetos e acompanha suas recompensas.

## O que você vê

- **Projetos criados** — total de projetos
- **Projetos enviados para clientes** — quantos foram compartilhados
- **Recompensa estimada** — valor potencial de projetos acessados por clientes
- **Recompensa efetivada** — valor real de compras realizadas

## Dentro de cada projeto

Clique em um projeto para ver:
- Informações (nome, cliente, data, capa)
- Ambientes e produtos especificados
- Valor total e recompensa estimada
- Categorias de produtos (gráfico)
- Social (curtidas, comentários, visualizações)
- Histórico de ações

## Criar novo projeto

Clique em **"Novo projeto"** no Dashboard. Dê um nome e comece a adicionar cenas e produtos.
