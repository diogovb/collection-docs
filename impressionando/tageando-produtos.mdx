# Tageando produtos na imagem

A Identificação permite marcar cada produto direto no render. Sua cliente vê exatamente o que é cada item — e pode comprar.

## Como tagear

<Steps>
  <Step title="Vá para a aba de Especificação no Studio">
    (Atualmente chamada "Especificação" no menu lateral.)
  </Step>
  <Step title="Clique na imagem">
    Clique no ponto onde o produto está.
  </Step>
  <Step title="Selecione o produto">
    Escolha qual produto da lista vincular àquele ponto.
  </Step>
  <Step title="Tag numerada aparece">
    Uma tag (1, 2, 3...) fica sobre a imagem, vinculada ao produto.
  </Step>
</Steps>

## Dicas

- Tageie **antes** de compartilhar com a cliente
- Use **"Ocultar/mostrar marcações"** para alternar a visibilidade
- No **Modo apresentação**, as tags ficam visíveis em tela cheia
