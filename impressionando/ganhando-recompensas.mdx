# Ganhando recompensas 💰

Sua cliente comprou pelo link que você compartilhou? Você ganha **5% do valor de cada produto**.

## Como funciona

1. Você especifica e compartilha o projeto
2. A cliente acessa e compra
3. Você recebe **5% de cada produto comprado**
4. Após **14 dias** (prazo de devolução), o valor fica disponível
5. Emita uma **NF** e receba via **PIX**

## Acompanhe no Dashboard

- **Recompensa estimada** — valor potencial (projetos acessados)
- **Recompensa efetivada** — valor real (compras concluídas)
- **Barra de progresso** por projeto

## Dica final

Quanto mais completa a especificação e mais bonita a apresentação, mais a cliente compra pelo seu link. Cada detalhe conta.
