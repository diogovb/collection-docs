# Como sacar recompensas

## Prazo

As recompensas ficam disponíveis para saque **14 dias após a compra** da sua cliente. Esse é o prazo de devolução — se a cliente devolver o produto, a recompensa é cancelada.

## Processo

1. Aguarde os 14 dias após a compra
2. Emita uma **Nota Fiscal** no valor da recompensa
3. Envie a NF para o Collection
4. Receba via **PIX**

<Info>
O processo de saque é manual por enquanto. Após enviar a NF, o pagamento via PIX é feito pela equipe do Collection.
</Info>
