# Recompensas — 5% por compra

Quando sua cliente compra um produto pelo link que você compartilhou, você ganha **5% do valor do produto**.

## Como funciona

1. Você especifica os produtos no projeto
2. Compartilha com a cliente via **"Enviar"**
3. A cliente acessa, vê os ambientes e compra
4. Você recebe **5% de cada produto comprado**

## Acompanhamento

No Dashboard, você vê:

- **Recompensa estimada** — valor potencial de projetos que a cliente acessou
- **Recompensa efetivada** — valor real de compras concluídas

Dentro de cada projeto, a **barra de progresso** mostra quanto falta para seu objetivo de recompensa.
